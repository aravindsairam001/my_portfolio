Current Research & Projects

Applying theory to practice.

Drone-Based Human Tracking & 3D Reconstruction

Developing an autonomous drone system using ESP32 and YOLOv8 for real-time human tracking. The project also involves using drone footage for high-fidelity 3D model reconstruction of structures like bridges using tools like COLMAP and Meshroom.

ESP32YOLOv8OpenCVCOLMAP

Bridge Defect Detection & Visualization

Creating an AI pipeline to automatically detect and classify 19 types of structural defects from images using the DACL10K dataset. The ultimate goal is to visualize these defects by projecting semantic annotations onto the reconstructed 3D models.

PythonSegmentationBlenderAI/ML

Kitchen Cleaning Robot Simulation

Designed a ROS2 simulation featuring a UR5e/xArm robotic arm for an automated kitchen cleaning task. The robot performs scene segmentation via an RGB-D camera and executes precise circular cleaning paths within a basin using MoveIt for motion planning.
