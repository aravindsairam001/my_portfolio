An autonomous drone built on ESP-Drone firmware that uses an ESP32-CAM to stream video. A ground station running Python with YOLOv8 detects a person in the video feed and sends real-time control commands (Pitch, Roll, Yaw) back to the drone via UDP sockets to actively follow the target.
